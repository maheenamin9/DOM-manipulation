# DOM-manipulation-project

A simple project built using JS DOM, local storage comprising of following functionalities: 

* Fetched users from [github users API](https://api.github.com/users).
* Store fetched data on local storage.
* Fetched data can be displayed by pressing "fetch users" button.
* A bootstrap form is created to get input data.
* Form data is also stored on local storage.
* Form data can be displayed by pressing "Form records" button.
* A user can delete particular record as well as all records in one go and it will also be deleted from local storage.
* Futhermore, alerts are used in different situations (say if user tries to access form records that isn't present etc).
